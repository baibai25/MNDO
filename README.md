# MND-based-Over-sampling
Multivariate Normal Distribution based Oversampling

## Usage

First download [Keel datasets](http://sci2s.ugr.es/keel/datasets.php):

    MNDO
    └── Dataset
    └── Predataset
    ├── train.ipynb
    ├── pre_dataset.py
    └── Multivariate_os.py 


Preprocessing:
    $ python pre_dataset.py xxx.dat
